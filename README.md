# Checksum Matcher

An application to manage your digital files collection.

- Scan your folders using checksum databases (dat files) to find missing, incorrect
  or unused files.
- Rebuild your files to match the database into folders or archives
  (zip, uncompressed 7z, torrentzip or torrent7z).
- Save time by scanning multiple databases in a single operation.
- Cache the directory contents of your external drives as offline storage.
  - Explore your drives without needing to connect them.
  - Scan using either online storage or offline storage.
- Automate download of databases and other artifacts from redump.org.
