# Checksum Matcher

An application used to scan, rebuild or create checksum databases (dat files).

- Supports ClrMame dats in standard or xml format.
- Scan multiple databases at a time.
- Cache the directory contents of your external drives as offline storage.
  - Explore your drives without needing to connect them.
  - Scan can be configured to use regular folders from online storage or offline storage.
- Rebuild to folders, zip, uncompressed 7z, torrentzip or torrent7z.

## Requirements

- Windows 11
